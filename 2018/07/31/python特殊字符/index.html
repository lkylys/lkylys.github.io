特殊字符

题上的要求是，给了一个txt文件，里面有一些文字，要求我们读取出里面的文字，我们需要输入一句话，如果我们是所输入的话中存在文件中的那些字符，那么那个字就要变成*输出，不是的话就不变化输出。
下面是我所做的txt中的文字：

黑产，
网马，
暗链，
挖矿，

第一:如何读取文件里面的内容，并将整合成一串字符或者list。并且文件里面有“，”和换行“\n”，我就想如何将它们去。
第二：我看到这个题，首先我想到就是使用两个循环，遍历特殊字符对输入的每一个字符都要做比较。

解决掉这两个问题其他就简单了。

第一个问题：

import sys
m = []
print("请输入一段话：")
n = input()
with open("C:/Users/李浴淑/Desktop/filter.txt","r")as x :
s = x.read()
for i in s:
if i == "\n" or i == "，":
pass
else:
m.append(i)

首先我创建一个空list留着存放去掉换行和“，”的字符，接着就是读取txt文件，然后遍历txt文件内容，将不是换行和“，”的东西存放在已经准备好的空list中。

第二个问题：

for i in n:
for j in m:
s = 0
if i == j:
print("*",end = '')
s += 1
break
if s == 0:
print(i,end = '')

将我们输入的内容和列表中的内容对比，相同则输出“*”
这两个循环顺寻不能换，必须是输入的每一个字符都要和list中的所有元素作比较。也就是遍历list对输入的每一个字符。代码中第一个判断是判断，两个字符是否相等，第二个是判断，相等的字符手否需要二次输出。
